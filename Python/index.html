Base Types

Basic data types in Python include integers, floats, booleans, and strings.

# Integer
x = 42

# Float
y = 3.14

# Boolean
is_true = True

# String
message = "Hello, World!"

Container Types

Containers store multiple values. Common containers are lists, tuples, sets, and dictionaries.

# List
fruits = ["apple", "banana", "cherry"]

# Tuple
coordinates = (10, 20)

# Dictionary
person = {"name": "John", "age": 30}

# Set
unique_items = {1, 2, 3}

Identifiers

Identifiers are names given to entities in Python. They follow specific naming rules.

# Naming rules:
# - Must start with a letter (a-z, A-Z) or underscore (_)
# - Followed by letters, digits (0-9), or underscores
# - Case-sensitive
my_variable = 10

Variable Assignments

Variables are assigned using the equals sign (`=`).

# Single assignment
x = 10

# Multiple assignment
a, b, c = 1, 2, 3

Conversions

Data types can be converted using built-in functions.

# Converting to string
num = 42
str_num = str(num)

# Converting to integer
str_value = "10"
int_value = int(str_value)

# Converting to float
float_value = float(str_value)

Boolean Logic

Boolean values can be combined using logical operators.

# Logical operators
x = True
y = False

# AND
result = x and y  # False

# OR
result = x or y  # True

# NOT
result = not x  # False

Statements Blocks

Statements are grouped into blocks using indentation.

# Defining a block
if x > 10:
    print("x is greater than 10")
    print("This is inside the block")
print("This is outside the block")

Modules/Names Imports

Modules can be imported using the `import` statement.

# Importing an entire module
import math

# Importing specific functions
from math import sqrt, pi

# Using an imported function
result = sqrt(16)  # 4.0

Conditional Statements

Control the flow using `if`, `elif`, and `else`.

if x > 10:
    print("x is greater than 10")
elif x == 10:
    print("x is equal to 10")
else:
    print("x is less than 10")

Maths

Basic arithmetic operations and math functions.

# Arithmetic operations
a = 5
b = 3

sum = a + b          # Addition
diff = a - b         # Subtraction
product = a * b      # Multiplication
quotient = a / b     # Division

# Using math functions
import math
sqrt_value = math.sqrt(25)  # Square root

Loops

Loops in Python are used to iterate over sequences, such as lists, tuples, and strings.

# For loop
for i in range(5):
    print(i)

# While loop
i = 0
while i < 5:
    print(i)
    i += 1

Functions

Functions are defined using the `def` keyword.

# Function definition
def greet(name):
    return f"Hello, {name}!"

# Calling the function
message = greet("Alice")  # "Hello, Alice!"

String Operations

Common operations on strings include concatenation, slicing, and formatting.

# String concatenation
greeting = "Hello, " + "World!"

# String slicing
substring = greeting[0:5]  # "Hello"

# String formatting
name = "Alice"
formatted = f"Hello, {name}!"  # f-string

File Handling

Opening, reading, and writing files.

# Writing to a file
with open("file.txt", "w") as file:
    file.write("Hello, World!")

# Reading from a file
with open("file.txt", "r") as file:
    content = file.read()

Errors and Exceptions

Handling exceptions using `try` and `except`.

try:
    result = 10 / 0
except ZeroDivisionError:
    print("You can't divide by zero!")

List Comprehensions

Creating lists in a concise way.

# List comprehension
squares = [x**2 for x in range(10)]  # [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

Lambda Functions

Anonymous functions defined with `lambda` keyword.

# Lambda function
add = lambda x, y: x + y
result = add(5, 3)  # 8

Dictionaries and Sets

Key-value pairs and unique collections.

# Dictionary
person = {"name": "Alice", "age": 25}
print(person["name"])  # "Alice"

# Set
unique_numbers = {1, 2, 3, 3}
print(unique_numbers)  # {1, 2, 3}
